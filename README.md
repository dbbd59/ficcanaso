# FICCANASO

## Using the Lints

To use the lints add a dependency in your `pubspec.yaml`:

```yaml
dev_dependencies:
  ficcanaso: ^4.0.0
  dart_code_metrics: ^4.15.1
```

then, add an include in your `analysis_options.yaml`

```yaml
include: package:ficcanaso/analysis_options.yaml
```
